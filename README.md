# Remember - 你的智能提醒助手

一个简洁、智能的桌面提醒工具，会自动在合适的时间提醒你。

## 自动提醒机制

软件会在后台自动运行，无需你操心：

### 📢 30分钟前 - 消息提醒
- **自动发送系统通知**（屏幕右上角弹出）
- 状态栏显示提醒文字
- **无声音**（不打扰你工作）

### 🔔 10分钟内 - 声音提醒
- **自动播放提示音**（每2分钟一次）
- 状态栏变红色警告
- 窗口闪烁引起注意

### 💡 到期时间 - 弹窗提醒
- 弹出带灯泡图案的提醒窗口
- 显示原定时间和实际提醒时间
- 可选择"已完成"或"稍后提醒"

## 功能特点

- ✅ **全自动提醒** - 后台运行，到时间自动提醒
- ⏰ **延迟提醒** - 设置5:00，可选30分钟后（5:30）提醒
- 🎵 **自定义铃声** - 支持 MP3/WAV/OGG 音频文件
- 🔄 **重复任务** - 每天/每周/每月自动重复
- 💾 **本地存储** - 数据保存在本地，保护隐私
- 🐥 **可爱小鸡** - 第二天打开时，小鸡会问候你

## 安装要求

- Python 3.7 或更高版本
- Tkinter（Python 标准 GUI 库）

### 可选依赖

**Windows:**
```bash
pip install win10toast
```

**Linux:**
```bash
sudo apt-get install libnotify-bin pulseaudio-utils
```

**macOS:** 无需额外安装，使用系统自带工具

## 使用方法

### 1. 启动程序
```bash
python3 main.py
```

### 2. 添加任务
- 点击 "+ 新建任务"
- 输入任务标题
- 设置原定时间
- 选择延迟时间（可选）
- 选择是否重复
- 保存

### 3. 自动提醒
软件会在后台自动运行：
- 30分钟前 → 系统通知
- 10分钟内 → 声音提醒
- 到期时间 → 弹窗提醒

### 4. 设置铃声
- 点击 ⚙️ 齿轮按钮
- 选择"自定义"
- 选择音频文件
- 调节音量
- 保存

## 数据存储

所有任务数据保存在同目录下的 `tasks.json` 文件中。

## 文件结构

```
remember/
├── main.py           # 主程序
├── requirements.txt  # 依赖文件
├── README.md         # 说明文档
└── tasks.json        # 任务数据（运行时生成）
```

## 提醒示例

假设你设置了一个任务：
- **原定时间**: 14:00
- **延迟**: 30分钟

软件会自动：
1. 13:30 - 系统通知："还有30分钟"
2. 14:20 - 播放提示音："还有10分钟"
3. 14:28 - 播放提示音："还有2分钟"
4. 14:30 - 弹窗提醒："该做任务了！"
